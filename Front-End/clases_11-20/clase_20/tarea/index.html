JSON y ejercicios de json

📝 Ejercicio 1: Convertir un objeto a JSON

Tienes este objeto de usuario en JavaScript:

const usuario = {

nombre: "Pedro",

edad: 25,

pais: "Argentina"

};

🔄 Ejercicio 2: Convertir JSON a objeto

Tienes esta cadena JSON:

{

"producto": "Teclado mecánico",

"precio": 120,

"stock": true

}

👉 Convierte esta cadena a un objeto de JavaScript con JSON.parse() y muestra el resultado.

📋 Ejercicio 3: Recorrer datos en JSON

Tienes esta cadena JSON que representa una lista de frutas:

["🍎 Manzana", "🍌 Banana", "🍊 Naranja"]

👉 Convierte este JSON en un array de JavaScript y recórrelo con un bucle mostrando cada fruta en consola.
