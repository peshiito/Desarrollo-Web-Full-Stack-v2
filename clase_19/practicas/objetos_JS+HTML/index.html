⚡ Ejercicio 1: Mostrar datos de un objeto

Crea un objeto persona con propiedades como nombre y edad.

Agrega un botón en HTML que, al hacer clic, muestre los datos de la persona en pantalla

Boton para mostrar los datos

⚡ Ejercicio 2: Modificar un objeto desde un input

Crea un objeto usuario con una propiedad nombre.

Agrega un input en HTML para escribir un nuevo nombre y un botón que actualice el objeto con ese valor y lo muestre en pantalla.

⚡ Ejercicio 3: Cuenta bancaria

Crea un objeto cuenta con una propiedad saldo.

Agrega un input en HTML y dos botones:

  • Uno para depositar dinero en la cuenta.
  • Otro para retirar dinero.

El saldo debe mostrarse en pantalla y actualizarse. Si se intenta retirar más de lo disponible, muestra un mensaje de error.
